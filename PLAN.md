# Bendele Electric Website Plan

## Project Goal
Create a sleek, modern, and professional website for Bendele Electric.
Focus on SEO and clear business identity.

## Pages
1. **Home**: Overview, Services, Trust signals.
2. **Get a Quote**: Detailed form for service requests (Main CTA).
3. **Contact Us**: General contact info, map, simple contact form.
4. **About**: Company history, team, values.

## Tech Stack
- **Core**: HTML5, CSS3
- **Styling**: Tailwind CSS (For modern/sleek look)
- **Scripting**: Vanilla JavaScript
- **Form Handling**: EmailJS

## Design System
- **Color Palette**:
    - **Primary (Accent)**: `#438548` (Professional Green)
    - **Secondary (Dark)**: `#1F2937` (Charcoal/Slate for text)
    - **Backgrounds**: `#FFFFFF` (White) and `#F3F4F6` (Light Gray)
    - **Action/Hover**: `#366A3A` (Darker Green)
- **Typography**:
    - **Headings**: Poppins (Sleek, geometric sans-serif)
    - **Body**: Lato (Legible, friendly sans-serif)

## Functional Requirements
- **Quote Form**: Must send structured email to admin via EmailJS.
- **SEO**: Semantic HTML, meta tags, fast loading.
- **Responsive**: Mobile-first design.
